# Tugas-Akhir
Tugas Akhir - Mobile Automation 

## Need to be installed
Node JS<br>
Java SDK & set JAVA_HOME<br>
Android Studio & set ANDROID_HOME<br>
Appium Server<br>
Appium Inspector<br>
LDPlayer (Emulator)<br>

## How To run
```
  npm install
```
```
  npm run wdio
```
